■はじめに
WP-ImaGeoMap は WordPress の記事やページに Google Map と写真を組み合わせた情報を
表示しますプラグインです。

旅行で撮影した写真を利用した旅日記や、お気に入りのお店を紹介するなど、様々な利用
方法が考えられます。
